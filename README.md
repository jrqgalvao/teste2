# Pousada Lua Rosa

Primeira versão estrutural e visual do website da Pousada Lua Rosa, na Praia
do Rosa, Santa Catarina.

## Executar

Requer Node.js 22.13 ou superior.

```bash
npm install
npm run dev
```

Validação:

```bash
npm test
npm run lint
```

## Conteúdo editável

Links, WhatsApp, contato, acomodações e caminhos de imagens estão centralizados
no início de `app/page.tsx`. A página usa imagens provisórias do Unsplash; elas
devem ser substituídas pelas fotografias oficiais antes da publicação.

Este protótipo não inclui backend, motor de reservas, mapa interativo, CMS,
analytics ou hospedagem.
